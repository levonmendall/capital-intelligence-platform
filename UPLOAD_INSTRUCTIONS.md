# Sprint 2 Upload Instructions

Upload these files into the root of the existing `ai-chief-investment-office` repository while preserving the folders. Replace `app.py`, `core/database.py`, `tests/conftest.py`, and `README.md` when prompted.

Then run:

```bash
python initialize.py
python run_intelligence.py
pytest -q
streamlit run app.py
```

The demonstration snapshot is neutral sample data. It is not represented as live market information.
